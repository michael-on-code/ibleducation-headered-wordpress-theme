IBL Education Headered Theme
========================

WordPress theme that includes the header (in Desktop and Mobile) 
of [ibleducation.com][1]

Requirements
------------

  * Wordpress Installation v5 or higher
  * PHP 7.0 or higher

Installation
------------

  * Insert these files in a folder (and name it ibleducation)
  * Zip the folder
  * Upload the theme zip file to into wordpress. In the administration go to 
  Appearances >> Themes >> Upload 


Usage
-----

A settings page is made available for setting :

* Site's Logo
* Additional Header Infos

Enjoy !

[1]: https://www.ibleducation.com
